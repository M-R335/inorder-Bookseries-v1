# Reading Order Books (InOrderBookSeries)

A comprehensive guide for book lovers to find the correct reading order for their favorite series and authors.

**Live Website:** [https://inorderbookseries.com/](https://inorderbookseries.com/)

## Features

- **Search**: Find authors, series, or books instantly.
- **Author Guides**: Complete bibliographies ordered chronologically.
- **Series Lists**: Correct reading orders for complex series.
- **SEO Optimized**: Built with Next.js 16 App Router for maximum performance and visibility.
- **Responsive Design**: Beautiful, premium UI that works on all devices.

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL (AWS RDS)
- **Deployment**: AWS Elastic Beanstalk

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Deployment

This project is configured for deployment on AWS Elastic Beanstalk (Node.js 20).
